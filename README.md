# 🍔 Driven Eats

🇺🇸 A simple food ordering interface built with HTML, CSS and JavaScript  
🔽 🇧🇷 Versão em português abaixo

---

## 🇺🇸 About the Project

Driven Eats is a small web application inspired by a snack bar ordering system.

The user can select one item from each category:

- Main dish
- Drink
- Dessert

After selecting all three items, the order button is enabled. The app then displays a confirmation screen with the selected products, individual prices and total amount. Once confirmed, the user is redirected to WhatsApp with a pre-filled order message.

This project was originally built as a study exercise while learning the fundamentals of web development. Because of that, some implementation choices were intentionally kept simple and close to the original learning context.

It is not intended to be a production-ready commercial application, but rather a portfolio project focused on practicing basic HTML, CSS and JavaScript concepts.

---

## 🚀 Features

- Select one product from each category
- Visual feedback for the selected option
- Enable order confirmation only after all required items are selected
- Calculate the total price
- Show an order confirmation modal
- Redirect to WhatsApp with a formatted message

# 🇧🇷 Driven Eats

Uma interface simples de pedidos para lanchonete, desenvolvida com HTML, CSS e JavaScript.

---

## 📖 Sobre o Projeto

Driven Eats é uma pequena aplicação web inspirada em um sistema de pedidos de uma lanchonete.

O usuário pode selecionar um item de cada categoria:

- Prato principal
- Bebida
- Sobremesa

Depois que os três itens são selecionados, o botão de fechar pedido é habilitado. Em seguida, a aplicação exibe uma tela de confirmação com os produtos escolhidos, os valores individuais e o total do pedido. Ao confirmar, o usuário é redirecionado para o WhatsApp com uma mensagem de pedido já preenchida.

Este projeto foi desenvolvido originalmente como exercício de estudo durante o aprendizado dos fundamentos de desenvolvimento web. Por isso, algumas escolhas de implementação foram mantidas simples e próximas do contexto original de aprendizado.

Ele não tem o objetivo de ser uma aplicação comercial pronta para produção, mas sim um projeto de portfólio focado na prática de conceitos básicos de HTML, CSS e JavaScript.

---

## 🚀 Funcionalidades

- Seleção de um produto por categoria
- Feedback visual para o item selecionado
- Liberação do botão de pedido apenas após selecionar os três itens
- Cálculo do valor total
- Tela de confirmação do pedido
- Redirecionamento para o WhatsApp com mensagem formatada

---

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- Ionicons
- Google Fonts

---

## 🎯 Objetivos de Aprendizado

Este projeto foi criado para praticar:

- Estruturação de páginas com HTML
- Estilização e layout com CSS
- Flexbox e seções com rolagem horizontal
- Manipulação do DOM com JavaScript
- Eventos de clique
- Estados condicionais na interface
- Manipulação básica de strings e números
- Redirecionamento para WhatsApp com parâmetros na URL

---

## 📌 Observações

Este projeto foi criado em uma fase inicial da minha jornada de aprendizado em desenvolvimento web.

Alguns pontos do código poderiam ser melhorados atualmente, principalmente em relação à escalabilidade, estrutura de dados, separação de responsabilidades e renderização dinâmica. Porém, a versão atual preserva intencionalmente a simplicidade da implementação original, pois o objetivo principal é demonstrar o processo de aprendizado e os fundamentos praticados na época.
