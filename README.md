## 🇺🇸 About the Project

Driven Eats is a small web application inspired by a snack bar ordering system.

The user can select one item from each category:

- Main dish  
- Drink  
- Dessert  

After selecting all three items, the order button is enabled. The app then displays a confirmation screen with the selected products, individual prices and total amount. Once confirmed, the user is redirected to WhatsApp with a pre-filled order message.

This project was originally built as a study exercise while learning the fundamentals of web development. Because of that, some implementation choices were intentionally kept simple and close to the original learning context.

---

## 🚀 Features

- Select one product from each category  
- Visual feedback for selected items  
- Conditional button enabling  
- Total price calculation  
- Order confirmation modal  
- WhatsApp redirection with formatted message

## 📖 Sobre o Projeto

Driven Eats é uma pequena aplicação web inspirada em um sistema de pedidos de lanchonete.

O usuário pode selecionar um item de cada categoria:

- Prato principal  
- Bebida  
- Sobremesa  

Após selecionar os três itens, o botão de fechar pedido é habilitado. A aplicação exibe uma tela de confirmação com os produtos escolhidos, os valores individuais e o total. Ao confirmar, o usuário é redirecionado para o WhatsApp com uma mensagem já preenchida.

Este projeto foi desenvolvido como exercício de estudo durante o aprendizado de desenvolvimento web, por isso mantém uma implementação simples.

---

## 🚀 Funcionalidades

- Seleção de um item por categoria  
- Feedback visual de seleção  
- Liberação condicional do botão de pedido  
- Cálculo do valor total  
- Tela de confirmação  
- Redirecionamento para o WhatsApp
